# Post Templates

이 폴더는 블로그 글 초안 템플릿 모음입니다.

사용 방법:

1. 카테고리에 맞는 템플릿 파일 하나를 고릅니다.
2. 파일을 복사해서 `_posts/` 아래에 `YYYY-MM-DD-title.md` 형식으로 저장합니다.
3. `title`, `date`, `tags`, 본문 내용을 수정합니다.
4. 커밋 후 푸시하면 블로그에 반영됩니다.

예시:

- `templates/dev-template.md`
- `_posts/2026-04-07-rag-project.md`

주의:

- `categories` 값은 템플릿에 들어 있는 값을 그대로 유지하는 편이 안전합니다.
- 파일명은 영문 소문자와 하이픈 위주로 작성하세요.
- 태그는 3~5개 정도로 간단히 정리하는 편이 보기 좋습니다.
