---

## Push Main

### 상황

- feature 브랜치에서 작업 완료
- main을 최신 상태로 유지해야 한다.

### 해결 흐름

- git pull --rebase
- git push
 
### 포인트

- pull 할 때 rebase 사용 → 히스토리 깔끔
- 최신 main 기준으로 작업 정렬 후 push
